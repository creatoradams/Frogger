# Crossing The Code: A Frogger Game Implementation
A modern remake of the classic arcade game, created in Unity.

Overview
This project is a full recreation of Frogger, built using Unity and C#.
The objective is simple: guide the frog across dangerous roads and rivers to reach home safely.
Along the way, the game demonstrates Unity fundementals such as scene management, physics, collisions, animation, audio management, and UI scripting.

This project was developed for CSCI: 4490 Software Enginerring | Fall 2025

Features

2 Scenes
- Main Menu
- Gameplay (Frogger)
Obstacle Movement
- Cars
- Logs
- Lilly Pads
Custom Audio Manager
- Background Music
- Home Music
- SFX for hops, death, respawn, etc.
Lives & Scoring System
Win Detection & Game Over Logic
Unity Scene Management
Optimized for Windows executable build (can also run on MacOS)

Tech Stack
-Unity Version: 2022+
- Language: C#
- Platform: Windows (Standalone Build)

How To Run The Game | You do NOT need Untiy installed.
- Download the latest release from the "WindowsBuild" folder OR "MacOSBuild"
- Extract the zip file
- Run Frogger.exe
- Make sure the Frogger_Data folder stays next to the .exe
